Pocket Winsdow's

A Five Nights at Freddy's fan game
The official Five Nights at Winsdow Prequel Game

Make sure you check the official Pocket Winsdow's game in development by Izuwii and his co-developer Shady Wolf on Game Jolt! X3




This game will contain the following characters:

  • Scorched Winsdow
  • Licy The Wolf
  • Waddle
  • Patty the Mouse
  • Neo The Red Panda
  • Nom Noms
  • The Hat
  • Muffin
  • Fantasy Winsdow
  • Misty




    • Pocket Winsdow's Will have a lot of original features, such as:

    - Different Game Modes

  • Infinite nights that will set scores
  • Easter Egg Game Mode (can be unlocked at the 5th night)
  • (Upcoming modes)

    • - In game Coins

  • Game Coins, that you can get by playing minigames, finishing challanges, finishing puzzles, time with infinite nights, crates, and much more.
  • (There might be more ways in the future)

    • - Skin Packs

  • Animatronic skins! (These skins are different for each character)
  • Animatronic skin packs! (The pack means that there's a version of that specific skin for all the characters)
  • Door Skins! (You can literally even have skins on your doors)
  • Office Skins! (This one is already known)

    • (They may be more stuff if we get new ideas)


    All of those skins can be bought with IN GAME coins.

    The skins can be obtained with crates that you can buy in the game shop with IN GAME coins, also you can play with grapple machines and more to get special skins!
